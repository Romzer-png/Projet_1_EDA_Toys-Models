# 📊 Analyse de données – SQL & Power BI

## 📌 Contexte

Dans un environnement data-driven, les entreprises cherchent à exploiter leurs données pour mieux comprendre leur activité et améliorer leur prise de décision.

Dans ce projet, l’objectif est d’analyser des données issues d’une base SQL afin d’identifier des tendances et produire des indicateurs clés.

---

## 🎯 Objectifs

En tant que Data Analyst, la mission consiste à :

- Extraire et analyser des données via SQL  
- Identifier des tendances et KPI  
- Concevoir un dashboard interactif avec Power BI  

---

## ⚙️ Méthodologie

- Extraction des données (SQL)  
- Analyse exploratoire  
- Création de visualisations  
- Construction du dashboard  

---

## 📈 Résultats

- Mise en évidence des indicateurs clés  
- Visualisation claire des données  
- Support à la prise de décision  

---

## 🛠️ Outils

- SQL  
- Power BI  
- Jupyter Notebook
